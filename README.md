# Sistema NoSQL para Armazenamento Esparso de Eventos RTSP

## 📌 Tema do Projeto
Este projeto propõe o desenvolvimento de um sistema baseado em NoSQL para armazenamento esparso de eventos extraídos de streams de vídeo RTSP.

Ao invés de armazenar vídeos completos, o sistema registra apenas eventos relevantes detectados em tempo real, como movimento ou presença de objetos, reduzindo significativamente o volume de dados.

---

## 🚀 Diferencial do Sistema

O sistema apresenta os seguintes diferenciais:

- Armazenamento esparso orientado a eventos
- Redução de custo computacional e de armazenamento
- Estrutura flexível baseada em documentos (JSON)
- Escalabilidade para múltiplas câmeras
- Preparação para integração com modelos de aprendizado de máquina
- Base para análises futuras de visão computacional

---

## 🧠 Tecnologias (planejadas)

- Banco NoSQL
- Streams RTSP
- Processamento de vídeo (OpenCV / FFmpeg)
- Integração futura com Machine Learning
